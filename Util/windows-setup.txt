Installing Windows 10 from an Official ISO Image

Remove all USB external hard drives before installation; otherwise, the Windows 10 installation process may fail.

Boot the system into the WinPE environment.

Delete all partitions on the target disk to ensure that the disk is completely clean.
⚠️ This will erase all data on the disk.

Use the virtual CD/DVD drive provided by WinPE to mount the Windows 10 ISO image file, then double-click setup.exe to start the installation program.

During installation, create a temporary user account named user.

Log in using the user account and enable the Administrator account. Open Command Prompt, right-click and select Run as administrator, then execute:

net user administrator /active:yes

Log in using the Administrator account and delete the temporary user account.

Change the location of the Documents folder to D:\Documents, and the Downloads folder to D:\Downloads.

In Desktop Icon Settings, enable the following system icons: Control Panel, This PC, and Command Prompt.

Install the NVIDIA GPU driver and the Realtek HD Audio driver. Set the display resolution to 3840×2160, set text scaling to 100%, and adjust the system volume to 20%.

In Mouse Settings, set Pointer Options → Pointer speed to the maximum. Under Wheel → Vertical scrolling, set it to 15 lines at a time.

In Display Settings, configure multiple monitors. Position the auxiliary displays on both sides of the primary display and align them to the lower-left or lower-right corner of the primary display to prevent the mouse cursor from frequently moving out of the main display unintentionally.

Install the wireless network adapter driver and connect to Wi-Fi. In Network Connections, disable all network adapters except the wireless adapter so that only one wireless interface remains active.

Connect only to the router’s 5 GHz Wi-Fi network (SSID). Do not connect to mixed 5 GHz / 2.4 GHz networks, otherwise after waking from sleep the system may reconnect to the 2.4 GHz band, resulting in slower network speed.

In Windows Security → Virus & threat protection, disable Real-time protection.

Activate Windows 10 using KMSoffline (or the W10 Digital Activation function of KMS).

Install Sogou Input Method and set the candidate count to 9. Remove Microsoft Pinyin Input Method. In language preferences, add English (United States) and set the default keyboard layout to English – US keyboard.

Install the Monaco font.

Install the following software and send their shortcuts to the desktop:

WinRAR, UltraISO, Source Insight 3.5/4.0, Notepad++, VS Code, HashCheck Shell Extension, Google Chrome, Microsoft Edge, Acrobat Reader, WinDjView.

Set the browser zoom level to 100%. After restoring from a GHOST image, the zoom level may be adjusted as needed; Chrome and Edge will remember the zoom setting for each website.

Configure all .txt files to open with Notepad++ by default.

Set the Sound Scheme to No Sounds.

Disable all notifications in Notification Settings.

In System Properties → Advanced → Performance, select Adjust for best appearance, and uncheck Animate windows when minimizing and maximizing.

In Windows Security → Virus & threat protection → Exclusions, add the directory D:\.

Adjust File Explorer settings according to personal preference.

Adjust Taskbar settings according to personal preference.

Optionally change the computer name to Xu-Chiheng.

In Power Options, disable Sleep and disable Turn off display. Under Processor power management, set Minimum processor state to 100% (default is 5%).

Send the v2rayN shortcut to the desktop, launch it, and configure it to start automatically at system startup.

Send the Beyond Compare shortcut to the desktop.

Configure the environment variables CYGWIN_DIR, MSYS2_DIR, VS_DIR, HOME, and DATA_DIR.

During the initial installation, set HOME to E:\Note. This must be a real directory, not a symbolic link, to avoid contaminating %USERPROFILE% (C:\Users\Administrator).
After restoring from a GHOST image, set HOME to %USERPROFILE%, and create a symbolic link:

E:\Note → %USERPROFILE%

Ensure that virtualization (Intel VT-x / AMD-V) is enabled in BIOS/UEFI before enabling Hyper-V.

Open Control Panel → Programs and Features → Turn Windows features on or off, then disable Internet Explorer 11 and enable Hyper-V.

Reboot into the WinPE environment and create a system backup using GHOST.

After rebooting into Windows 10, calculate the SHA512 checksum of the GHOST image file and copy it to an external hard drive.

Install Office 2019, and activate it using KMSoffline (or KMS AAct Network → Office Activation).

Install the following software and send their shortcuts to the desktop:

VLC media player, FileZilla, WinSCP, Baidu Netdisk, Thunder (Xunlei), qBittorrent.

Install 360 Security Guard, and perform system checkup, virus scan, system repair, system optimization, and system cleanup.

Uninstall 360 Security Guard.

Reboot into the WinPE environment and create another system backup using GHOST.

After rebooting into Windows 10, calculate the SHA512 checksum of the GHOST image file and copy it to an external hard drive.




从 Windows 10 原版 ISO 镜像安装 Windows 10

在开始安装前，移除所有 USB 移动硬盘设备，否则可能导致 Windows 10 安装过程失败。

启动系统进入 WinPE 环境。

删除目标系统硬盘上的所有分区，以确保磁盘处于干净状态。
⚠️ 此操作将清除磁盘上的所有数据，请提前做好备份。

使用 WinPE 自带的虚拟光驱挂载 Windows 10 ISO 镜像文件，双击 setup.exe 启动安装程序。

在安装过程中创建临时用户账户 user。

使用 user 账户登录系统后，启用 Administrator 账户。打开“命令提示符”，右键选择“以管理员身份运行”，在命令窗口中执行：

net user administrator /active:yes

使用 Administrator 账户登录系统，并删除临时账户 user。

将“文档”文件夹位置修改为 D:\Documents，将“下载”文件夹位置修改为 D:\Downloads。

在桌面图标设置中启用以下系统图标：控制面板、此电脑、命令提示符。

安装 NVIDIA 显卡驱动和 Realtek 高清音频驱动；设置显示分辨率为 3840×2160，文本缩放比例为 100%；将系统音量调整至 20%。

在鼠标设置中，将“指针选项 / 移动速度”调整至最高；在“滚轮 / 垂直滚动”中设置为“每次滚动 15 行”。

在显示设置中配置多显示器布局。将主显示器两侧的辅助显示器位置调整至主显示器的左下角或右下角，以避免鼠标指针频繁意外移出主显示器。

安装无线网卡驱动并连接 Wi-Fi。在“网络连接”中禁用除无线网卡以外的所有网卡，仅保留一个无线网卡。

只连接路由器的 5 GHz 无线网络（SSID），不要连接 5 GHz / 2.4 GHz 混合网络，否则系统从休眠恢复后可能自动连接至 2.4 GHz 网络，导致网络速度下降。

在“Windows 安全中心 → 病毒和威胁防护”中关闭“实时保护”。

使用 KMSoffline（或 KMS 的 W10 Digital Activation 功能）激活 Windows 10。

安装搜狗输入法，并将“候选项数”设置为 9。删除微软拼音输入法。在语言首选项中添加“英语（美国）”，并在键盘设置中将默认键盘设为“英语 - 美式键盘”。

安装 Monaco 字体。

安装以下软件，并将快捷方式发送到桌面：WinRAR、UltraISO、Source Insight 3.5/4.0、Notepad++、VS Code、HashCheck Shell Extension、Google Chrome、Microsoft Edge、Acrobat Reader、WinDjView。

将浏览器缩放比例设置为 100%。从 GHOST 恢复系统后，可根据需要调整缩放比例；Chrome 和 Edge 会记住每个网站的缩放设置。

将所有 .txt 文件的默认打开程序设置为 Notepad++。

将声音方案设置为“无声”。

在通知管理中关闭所有通知。

在“系统属性 → 高级 → 性能”设置中选择“调整为最佳外观”，并取消勾选“在最大化和最小化时显示窗口动画”。

在“Windows 安全中心 → 病毒和威胁防护 → 排除项”中添加 D:\ 目录。

根据个人使用习惯调整文件资源管理器（File Explorer）设置。

根据个人使用习惯调整任务栏设置。

可选地，将计算机名修改为 Xu-Chiheng。

在电源选项中禁用休眠，并禁用自动关闭显示器。在“处理器电源管理”中，将“最小处理器状态”修改为 100%（默认值为 5%）。

将 v2rayN 快捷方式发送到桌面，并启动 v2rayN，设置为开机自动启动。

将 Beyond Compare 的快捷方式发送到桌面。

设置环境变量 CYGWIN_DIR、MSYS2_DIR、VS_DIR、HOME、DATA_DIR。在初始安装阶段，将 HOME 设置为 E:\Note（必须为实际目录，而不是符号链接），以避免污染 %USERPROFILE%（C:\Users\Administrator）。从 GHOST 恢复系统之后，需要将 HOME 设置为 %USERPROFILE%，并建立符号链接 E:\Note → %USERPROFILE%。

在启用 Hyper-V 之前，请确保已在 BIOS/UEFI 中开启 CPU 虚拟化功能（Intel VT-x / AMD-V）。

在“控制面板 → 程序和功能 → 启用或关闭 Windows 功能”中，禁用 Internet Explorer 11，并启用 Hyper-V。

重启进入 WinPE 环境，并使用 GHOST 备份系统。

重新启动进入 Windows 10 后，计算 GHOST 镜像文件的 SHA512 校验值，并将该文件复制到移动硬盘。

安装 Office 2019，并使用 KMSoffline（或 KMS 的 AAct Network → Office Activation）激活 Office 2019。

安装以下软件，并将快捷方式发送到桌面：VLC media player，FileZilla，WinSCP，百度网盘，迅雷，qBittorrent。

安装 360 安全卫士，执行系统体检、病毒查杀、系统修复、系统优化以及系统清理操作。

卸载 360 安全卫士。

重启进入 WinPE 环境，并使用 GHOST 再次备份系统。

重新启动进入 Windows 10 后，计算 GHOST 镜像文件的 SHA512 校验值，并将该文件复制到移动硬盘。
